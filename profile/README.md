## 👋 Who We Are

We're **Nirmini Development** (Or just "Nirmini" for short) - an indie technology and game development studio. We build Discord tooling, narrative-driven games, and software that just works.

---

## 🛠️ What We're Working On

# <img src="https://github.com/Nirmini/.github/blob/main/assets/Nova-V3-S-T.png?raw=true" height="50" align="middle" /> [Nova](https://nirmini.dev/Nova)
A multi-purpose Discord bot and moderation platform built to manage communities with ease.

| Repo | Description |
|------|-------------|
| [Novacore](https://github.com/thatWest7014/Novacore) | Lightweight Discord bot base to build off of |
| [NovaAPI](https://github.com/Nirmini/NirminiAPIs) | API powering Nova's cross-platform communication |
| [NovaBot](https://github.com/Nirmini/Nova) | The fully built Discord bot for communities and moderators |
| [Novaworks](https://github.com/Nirmini/Novaworks) | A moderation platform for Roblox that actually works |
